## PREPARING STEPS
#### 1. Install venv:
```shell
$ python -m pip install virtualenv
$ virtualenv venv
```

#### 2. Activate venv:
```shell
$ source venv/bin/activate OR . venv/bin/activate
```

#### 3. Install requiements:
```shell
$ pip3 install -r requirements.txt
```
